# v0.1.0-alpha.6x-wave6

Wave 6 backlog convergence and canon lock release.

## Scope

Implemented items:
- C0: Canon truth-sync across ADRs, whitepaper, roadmap, and backlog execution docs.
- C1: Behavior-preserving `artifacts.rs` decomposition into focused modules.
- C2: Behavior-preserving `report.rs` decomposition into focused modules.
- C3: Compatibility/regression lock tests for module split and schema/read-path stability.

Out of scope (explicit):
- New runtime features.
- New `graph.json` schema fields.
- Deferred ADR-0017 fields (`op_hash`, `resources`, `cache_policy`, `materialization_policy`) implementation.

## What Changed

### C0 — Canon Truth-Sync
- Updated ADR implementation status notes for Wave 5 + Wave 6 reality.
- Added explicit ADR-0017 deferment note for remaining schema fields.
- Added explicit ADR-0017 hash policy note that `execution_hint` is excluded from `node_def_hash`.
- Added ADR-0018 trust-path note: `UpdateTransform` -> `TransformAuditV0` -> `UnsafeReasonV0::UnsafeExtension`.
- Updated whitepaper status/evidence text to match implemented artifact/dataops behavior.
- Updated roadmap/backlog docs to mark A6X-06 complete.

### C1 — Artifacts Module Decomposition (Behavior-Preserving)
`swarm-torch/src/artifacts.rs` was split into:
- `swarm-torch/src/artifacts/mod.rs`
- `swarm-torch/src/artifacts/bundle.rs`
- `swarm-torch/src/artifacts/sink.rs`
- `swarm-torch/src/artifacts/session.rs`
- `swarm-torch/src/artifacts/io.rs`
- `swarm-torch/src/artifacts/tests.rs`

Behavior and contract invariants preserved:
- Bundle layout and file names unchanged (`manifest.json`, `graph.json`, `datasets/*.json`, `*.ndjson`).
- `RunEventEmitter for RunArtifactSink` validation path unchanged.
- Materialization trust propagation and unsafe-reason semantics unchanged.

### C2 — Report Module Decomposition (Behavior-Preserving)
`swarm-torch/src/report.rs` was split into:
